# Cognitive Overload Signal Bloom

prev_id: [[cognitive-overload-signal-bloom-0c92c6e8]]

## Context

Reseeded from a doppl leaf — the unlock becomes the seed of a new island.

## Synopsis

Implementing synchronized, low-intensity visual stimuli across the yacht's open deck that are designed not to be distracting, but to maintain a constant baseline state of high cognitive load for any human observer (or simulated input processing unit). This forces observers into a perpetual sub-conscious filtering state.
